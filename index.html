Overview

  
The Affiliates Currency Manager is a tool for managing the supported digital currencies and countries for affiliates. It allows for the addition or removal of currencies and the addition of supported countries based on affiliate names.


  
Usage Instructions

  
    
    
  1. Affiliates: List the affiliates to update, separated by commas. Leave blank to apply changes to all affiliates.
    2. 
    
  2. LP Data: Input the JSON object containing affiliate data in the LP (JSON) field.
    3. 
    
  3. Currency Modifications: To modify currencies, check the appropriate box and enter the currencies as a JSON array.
    4. 
    
  4. Country Additions: To add countries, check the "Add Supported Countries" box and provide the countries as a JSON array.
    5. 
    
  5. Submit: Click "Update Affiliates" to apply changes. The result will be displayed in the Result section.
    6. 
    
  6. Copy Result: Use the "Copy Result" button to copy the updated LP data.
